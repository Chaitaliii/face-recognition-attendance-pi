# face-recognition-attendance-pi
Face recognition–based attendance system using Raspberry Pi and OpenCV.
## Goal
Build a Raspberry Pi–based face recognition attendance system using OpenCV.

## Hardware
- Raspberry Pi 4 (2GB)
- Pi Camera / USB Camera

## Software
- Raspberry Pi OS
- Python 3
- OpenCV
